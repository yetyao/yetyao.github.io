mysql

mysql-函数-1

1. if

IF(expr,v1,v2)
如果表达式 expr 成立，返回结果 v1；否则，返回结果 v2

2. ifnull

IFNULL(v1,v2)
如果 v1 的值不为 NULL，则返回 v1，否则返回 v2。

3. last_insert_id

LAST_INSERT_ID()
返回最近生成的 AUTO_INCREMENT 值

4. version

version()
返回mysql版本

5. isnull

isnull(null)
判断是否为null 是返回1 否返回0
