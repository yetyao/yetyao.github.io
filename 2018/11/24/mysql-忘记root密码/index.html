mysql 忘记root密码

mysql 版本 8.0.12

  1. 修改 /etc/my.cnf
    添加 skip-grant-tables
  2. systemctl restart mysqld
  3. mysql -u root -p (root身份登录)
  4. use mysql
  5. flush privileges
  6. alter user ‘root‘@’localhost’ IDENTIFIED BY ‘mypassword’;
